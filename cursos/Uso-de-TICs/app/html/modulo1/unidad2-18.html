Crear Carpetas y Archivos

La información en una computadora se organiza en archivos y carpetas. Cuando se elabora una carta y se guarda en la computadora se está creando un archivo y esa carta o archivo lo podemos colocar dentro de una carpeta.

Las carpetas se crean muy fácilmente y no existe un límite sobre el número de carpetas que se pueden tener.

El nombre de sus archivos o carpetas deben:

  1. Contener menos de 225 caracteres, incluyendo espacios.
  2. No pueden contener los siguientes caracteres: \ / : * ? " <> |.
  3. No utilizar ciertos nombres como: com1, com2, clock$, con, lpt1, prn, nul.

Estos nombres no pueden asignarse a un archivo o carpeta, ya que el sistema operativo los reserva para designar dispositivos específicos que usa la computadora.

Existen procedimientos muy sencillos para poder crear carpetas en Windows. Los más usados son a través de:

  • Menú Archivo
  • Menú Contextual
  • Punto Inicial

  1. Abra el Explorador de Windows
  2. Seleccione la opción de Nueva Carpeta (New Folder) en la barra que contiene el título del Explorador.
  3. La carpeta se creará automáticamente en el lugar que se haya elegido.
  4. Inmediatamente escriba el nombre de la carpeta recién creada.
  5. Haga clic fuera de la carpeta para deshabilitar esta función.

Los pasos para crear una carpeta con el menú contextual, son los siguientes:

  1. Ubique el ratón en el lugar físico donde desea crear la nueva carpeta.
  2. Haga clic con el botón derecho, para desplegar el menú contextual.
  3. Elija la opción Nueva Carpeta (New Folder).
  4. La carpeta se habrá creado; enseguida asigne un nombre para la carpeta.

La creación de la carpeta se ha vuelto muy sencilla en Windows 10 con el siguiente método:

  1. En el menú de Punto Inicial (Home), seleccione el botón de Nueva Carpeta (New Folder).
