¿Qué es el control biológico?

Esta debe ser la primera pregunta que debemos hacernos, si deseamos utilizar este método de control de plagas, que es más seguro para las personas, para los consumidores y para el medio ambiente.

El término control biológico fue usado por primera vez por Harry Smith, en 1919, para referirse al uso de enemigos naturales para el control de plagas agrícolas. El número y el tipo de enemigos naturales que se utilizan para controlar plagas agrícolas se ha incrementado desde entonces, por lo que ahora se dice que el control biológico es el uso de organismos vivos como agentes de control de plagas.

Una plaga puede ser controlada de varias maneras, por ejemplo si hay ratones en una casa o en una bodega, podemos usar:

Control Mecánico

Control Químico

Control Biológico

En este caso, el ratón, que es una plaga doméstica, tiene como enemigo natural al gato, el cual a mediano plazo nos controlará a los ratones que nos están afectando.

Enemigos naturales de las plagas

De esta misma forma las plagas agrícolas (como la gallina ciega, el gusano cogollero, el gusano del elote, etc) tienen enemigos naturales, los cuales se encuentran en los campos, en números más o menos reducidos. Para aprovecharlos se necesita conocerlos y protegerlos. También existen lugares en que se reproducen estos organismos benéficos, los cuales pueden ser adquiridos y liberados en nuestros campos para controlar plagas.
