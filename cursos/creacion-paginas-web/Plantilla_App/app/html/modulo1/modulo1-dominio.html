Tema 3 ¿Qué es un Dominio y para qué me sirve?

¿Qué es un Dominio? Al igual que su casa, un Sitio Web tiene una dirección, esta dirección generalmente es numérica, pero para el usuario no sería agradable o sencillo el relacionar su Sitio Web con una serie de números, por lo que se creó el término Dominio en la Web.

El Dominio en la Web es un nombre que se le asigna a una dirección IP o ubicación en un servidor donde se encuentra nuestro Sitio Web, y se emplea con nombres referentes al contenido de la Página.

Piense que tiene un Sitio Web, y le dijeran que su dirección IP es 123.45.678.910 y que tiene que recordar cada número para poder decirle a los usuarios que ahí podrán encontrar lo que necesitan, ¿verdad que no es sencillo?, en cambio qué le parece decir, le invito a que visite mi Sitio Web en www.esmipagina.com, ¡mucho mejor! ¿Cierto?

Un Dominio es la dirección donde se ubica mi Página Web, el cual debe ser único e irrepetible.

¿Qué es el URL?

El URL es un término llamado así por sus siglas en inglés Uniform Resource Locator.

El URL es un formato estándar para nombrar recursos en la Web, formado por un esquema y un dominio.

¿Para qué me sirve el Dominio?

El Dominio sirve para recordar fácilmente la dirección donde se encuentra el Sitio Web y al mismo tiempo facilitar la búsqueda a los usuarios que deseen acceder en Internet a una dirección. Ejemplos de Dominios:

  • www.elnorte.com
  • www.nl.gob.mx
  • www.cca.org.mx
  • www.delanaturalezaatumesa.mx/
  • www.itesm.mx
  • www.facebook.com

Obtener un dominio

1. Definir cómo se llamará su Sitio Web (ej. www.misitioweb.com)

2. Consultar la disponibilidad de su nombre de Sitio Web. Puede consultarlo en:

  • http://www.nic.mx/ si es un dominio .mx
  • http://www.whois.net/ si es un dominio .com o de cualquier tipo.

3. Localizar en Internet un proveedor de Hospedaje o Alojamiento Web que le venda el dominio.

Generalmente el alojamiento Web tiene un costo por el que deberá pagar, pero en otros lugares se ofrece gratuitamente, como lo veremos más adelante.
