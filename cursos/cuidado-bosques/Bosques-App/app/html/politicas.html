Políticas

Este curso forma parte de nuestra oferta educativa en la modalidad “Por ti mismo” (gratuito), por lo tanto, no tienes un tutor asignado para evaluar las actividades.

Tú decides cuándo y dónde acceder, permitiéndote elaborar un horario y ritmo de estudio propio, de acuerdo a tus circunstancias.

Se recomienda tener una buena organización, para realizar las actividades que contiene el curso, ya que fueron diseñadas para facilitar el proceso de aprendizaje.

Aplica lo aprendido para que determines tus conocimientos adquiridos.
