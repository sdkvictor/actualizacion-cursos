3. Power point

Presentación

¿Cómo vamos hasta este momento?

Vamos a comenzar con otro tema muy importante de las funcionalidades del Office, el uso del presentador electrónico o mejor conocido como Power Point.

Una presentación electrónica sirve para poner de forma gráfica alguna información sobre: Una tarea de la escuela, un catálogo de productos, una presentación para el trabajo, etc.
