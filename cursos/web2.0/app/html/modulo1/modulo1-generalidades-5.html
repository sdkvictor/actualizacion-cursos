Modulo 1 - Generalidades

Para ser más especificos, se explica a detalle cada una de las principales características de la Web 2.0 en la siguiente tabla.

Haz clic en cada una de las características para ver su descripción



1. La web es una plataformas

Hemos pasado de un Software inestable en nuestras computadoras a servicios de software que son accesibles online.

3. La web es simple

Facilita el acceso a los servicios web a través de pantallas más agradables y fáciles de usar.

5. La web es social

Las personas crean la web mediante la socialización y el movimiento gradual de los miembros del mundo físico hacia el mundo online.

7. La web es combinable

La expansión de códigos para poder modificar las aplicaciones web. Esto permite que los individuos que no son profesionales puedan combinar diferentes aplicaciones para crear nuevas.

9. La web está en nuestras manos:

El aumento de la organización de la información enfatiza el uso amistosos de la misma a través de los enlaces. Gracias al fenómeno social del etiquetado cada vez más fácil acceder a la información.

2. La web es funcionalidad

La Web ayuda en las transferencia de información y servicios desde páginas web.

4. La web es ligeras

Los modelos de desarrollo, los procesos y os modelos de negocio se vuelven ligeros. La ligereza está asociada con la habilidad para compartir la información y los servicios de forma fácil y hacerlo posible a través de la implementación de intuitivos elementos modulares.

6. La web es flexible

El software se encuentra en un nivel más avanzado, ya que este nivel, permite los accesos a contenidos digitales a los que antes no se podía llegar.

8. La web es participativa

La web 2.0 ha adoptado una estructura de participación que alienta a los usuarios a mejorar la aplicación mientras la utilizan, en lugar de mantenerla rígida y controlada

¿Cómo ven?, ¿muy interesante no?. ahora sabemos que muchas veces empleamos estas características y ni sabiamos que ya estabamos trabajando bajo la Web 2.0.

Web 1.0

En la Web 1.0 una persona (Webmaster) colocaba la información en un sitio web y todos los usuarios consultaban, siendo el webmaster el responsable de lo publicado. Los usuarios solo sabían que se actualizó dicha información cuando ingresaban y veían la diferencia o bien si se les informaba a través de un correo electrónico o cualquier medio digital.

Web 2.0

En la Web 2.0 no sólo una persona coloca información, sino un grupo de usuarios puede publicar o construir la información que está en los sitios web a través de diferentes herramientas, de tal forma que si existe una actualización por cualquier usuario se le avisara a los demás de forma electrónica a través de un RSS de forma automática.
