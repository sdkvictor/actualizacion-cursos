Reading Comprehension

Introduccion

Esta seccion esta disenada para desarrollar tus habilidades de lectura y comprension del idioma ingles. En este momento sera mas facil para ti entender la lectura, ya que se ha utilizado el metodo se comprender el significado del contexto sin utilizar algun metodo de traduccion gramatical.

Lee el siguiente material y contesta las preguntas.

Describing Food

Un adjetivo puede decirte como luce, se siente, huele, suena o sabe algo.

Example:
"These tacos are delicious. The meat is spicy and tender. The shell sounds hard and crunchy. These cheese is soft and the cilantro is aromatic. The chilies are hot and tasty. I really like tacos."

EXAMPLES OF WORDS THAT DESCRIBE FOOD
Delicious / Deliciosa Tender / Tierno, blando
Crunchy / Crujiente Aromatic / Aromático
Tasty / Sabroso, apetitoso Sour / Agrio, ácido
Cold / Fría Sticky / Pegajoso
Spicy / Condimentada Hot / Caliente
Soft / Suave Salty / Salado
Sweet / Dulce Bitter / Amargo
Warm / Calientita, tibia
